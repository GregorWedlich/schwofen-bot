## Getting Started

### Prerequisites

- Node.js
- PostgreSQL
- Telegram Bot Token
- Telegram Channel ID
- Configure environment variables

### Installation

1. Install dependencies: `npm install` or `pnpm install`
2. Generate Prisma Client: `npx prisma generate`
3. Start the development server: `npm run dev` or `pnpm dev`
